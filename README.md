# 智能工作日志编辑器 (WorkLog Editor) 

 一款跨平台桌面应用，用于高效记录每日工作日志，并集成 DeepSeek & Ollama AI 助手，支持智能工具调用。

 ##  主要功能 

 -  **Markdown 编辑**：支持实时预览、字数统计、自动保存。
 -  **智能 AI 助手**：
    - **多模型支持**：支持在线 DeepSeek API 及本地 Ollama 私有化模型。
    - **Tool Use (工具调用)**：AI 可以直接读取您的日志、搜索内容、统计时长，甚至直接为您草拟新日志。
    - **智能总结**：一键生成日报、周报或项目阶段总结。
 -  **全文搜索**：支持标题与内容的关键词快速检索。
 -  **导入/导出**：支持导出为 Markdown 文件或 ZIP 压缩包，方便备份与分享。
 -  **响应式设计**：完美适配不同尺寸屏幕，支持深色/浅色模式。

 ##  快速开始

 ### AI 配置指南

 1. **DeepSeek (推荐)**: 
    - 在设置中填入您的 API Key 即可。
 2. **Ollama (本地私有)**: 
    - 确保本地已启动 Ollama 服务。
    - 在应用设置中切换供应商为 "Ollama"。
    - 刷新并选择您已下载的模型（如 `llama3`, `qwen2` 等）。

 ### 常用功能

 - **对话助手**: 点击右侧 AI 图标，尝试输入 “帮我总结下本周的工作” 或 “搜索关于项目 A 的日志”。
 - **数据管理**: 通过侧边栏右上角的菜单，可以进行批量导出或 Markdown 导入。

 ##  快捷键 

 | 操作 | 快捷键 | 
 |------|--------| 
 | 新建日志 | `Ctrl/Cmd + N` | 
 | 保存当前日志 | `Ctrl/Cmd + S` | 
 | 聚焦搜索框 | `Ctrl/Cmd + Shift + F` | 
 | 切换 AI 面板 | `Ctrl/Cmd + J` (或点击按钮) | 

 ##  数据安全 

 所有日志数据及 AI 对话历史均加密存储在本地。使用 Ollama 时，所有 AI 处理均在本地完成，无需联网。
